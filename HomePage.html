What's and Why VPC

AWS offers several services to fully and seamlessly integrate your local resources with the cloud. One of those services is the Amazon Virtual Private cloud. Running applications in cloud will be completely transparent to end users. Amazon Virtual Private Cloud (Amazon VPC) is a virtually isolated network in the cloud, which resembles a traditional network in a corporate datacenter.

Amazon's Definition for AWS VPC:

"Amazon Virtual Private Cloud (Amazon VPC) enables you to launch Amazon Web Services (AWS) resources into a virtual network that you've defined. This virtual network closely resembles a traditional network that you'd operate in your own data center, with the benefits of using the scalable infrastructure of AWS."[1]
